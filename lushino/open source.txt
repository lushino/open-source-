Number of Freedom
-Freedom 0: The freedom to run the
program, for any purpose
-Freedom 1: The freedom to study how
the program works, and adapt it to your
needs. Access to the source code is a
precondition to this.
-Freedom 2: The freedom to redistribute
copies so you can help your neighbor.
-Freedom 3: The freedom to improve the
program, and release your
improvements (and modified versions in
general) to the public, so that the whole
community benefits. Access to the
source code is a precondition for this.
